#0	<a href=""></a>

This tag permits to create some links. It can be link for a web
or for an other file on the machin. This is the parameter href=""
that contains the link.
For example : 2# 
<!DOCTYPE>
<html>

    <a href="https://www.google.com"> Google here. Click to charge the page !! </a>
    <p> Or load my personal page : <a href="text.html"> HERE </a> </p>
  </body>
</html>
