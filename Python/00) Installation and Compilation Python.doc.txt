				INSTALLATION

This is simply :
#1 [ sudo apt-get install python3.8 ]

For installing tkinter :
#1 [ sudo apt-get install python3-tk ]

For installing PIP :
#1 [ sudo apt-get install python3-pip ]

For installing discord.py :
#1 [ python3.8 -m pip install -U discord.py ]

For installing dotenv :
#1 [ sudo apt-get install python3-dotenv ]

For installing Pygame :
#1 [ sudo apt-get install python3-pygame ]
